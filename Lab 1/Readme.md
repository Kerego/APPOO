#Lab 1 Bejenari Marian
###Framework: C#/.NET Core

####Usage
#####Clone Repository
```bash
git clone https://github.com/Kerego/APPOO.git
```
#####Open Lab 1 directory
```bash
cd APPOO/"Lab 1"
```
#####Run the application  
```bash
Standalone/"Lab 1" -i input.txt -n 1000 -o output.txt
```
